Introduction:
---------------------
In a Temple University classroom environment, a chatbot is created using advanced language models such as ChatGPT/Langchain.The chatbot was implemented on Slack and is designed to generate automated explanations to assist students in answering their inquiries.
The bot is capable of engaging in natural-sounding conversations and providing coherent and relevant responses.
additionally, a chat channel was developed in Slack APP, similar to a classroom where a professor teaches a group of students, enabling multiple people to interact with the chatbot and receive educational responses to their questions

Set up manual:
----------------------
go thru Slack_app_manual.pdf to generate token keys and set up slack bot
next, set up app.py script to run the application,please follow the instructions of slack_bolt_python_setup.pdf
